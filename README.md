# Sistema Cobranca Spring Boot
Sistema simples de cobranca usando spring boot 

## Tecnologias usadas:
 - Spring Boot 
 - Api Rest
 - Java
 - Thymeleaf
 - Html , Javascript , CSS e Bootstrap
 - Swagger( para documentação)
 - JUnit e EasyMock ( testes unitários )
 - JPA / Hibernate(Pesistencia)
 - H2 (Banco de Dados)
 - Maven
